# Trading Risk Calculator

A responsive position-sizing calculator for Indian intraday trades and crypto.

## Features

- Risk-based sizing from a maximum loss amount
- Capital and leverage sizing
- Long and short trades
- Stop-loss and target by percentage or price
- Estimated Indian intraday or crypto charges
- Net profit, loss, and risk/reward results
- Light and dark themes with offline PWA caching

Open `index.html` directly, or serve the folder with any static web server.
